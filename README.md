# Excel 数据智能分析工具

一个基于 Python 的办公自动化工具，支持拖拽上传 Excel 文件，自动进行多维数据统计分析，并生成专业的统计报表。

## ✨ 功能特性

- **拖拽上传**：支持将 Excel 文件直接拖入界面
- **自动分析**：批量解析 Excel 数据，进行部门/项目/状态多维统计
- **智能报表**：自动生成包含5个工作表的统计结果 Excel
- **美观界面**：卡片式设计，蓝色主题，状态实时反馈

## 📊 生成的报表内容

| 工作表 | 内容 |
|--------|------|
| 整体概况 | 总记录数、总工时、部门数、项目数、状态分布 |
| 部门统计 | 各部门条目数、工时、完成率 |
| 项目统计 | 各项目条目数、工时、完成率 |
| 待办事项 | 自动提取进行中/已延期/待开始任务 |
| 原始明细 | 完整原始数据备份 |

## 🚀 快速开始

### 环境要求

- Python 3.7+
- Windows 操作系统

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动程序

**方式一：双击启动（推荐）**

直接双击 `启动Excel分析工具.bat` 文件。

**方式二：命令行启动**

```bash
python gui_analyzer.py
```

### 使用步骤

1. 打开程序
2. 将 `.xlsx` 文件拖入拖拽区域（或点击「选择文件」按钮）
3. 确认输出位置（可点击「更改」自定义）
4. 点击「开始分析」按钮
5. 等待完成，结果文件自动保存

```

└── .gitignore               # Git忽略规则
```

## 🛠️ 技术栈

- **GUI**: tkinter + tkinterdnd2（拖拽支持）
- **Excel处理**: openpyxl
- **样式**: ttk.Style 自定义主题

## 📝 Excel 数据格式要求

输入文件需包含以下列：

| 列名 | 说明 |
|------|------|
| 序号 | 记录编号 |
| 日期 | 工作日期 |
| 部门 | 所属部门 |
| 员工 | 负责人姓名 |
| 项目名称 | 项目名称 |
| 工作内容 | 工作描述 |
| 完成状态 | 已完成/进行中/待开始/已延期 |
| 工时(h) | 投入工时 |
| 备注 | 其他信息 |
